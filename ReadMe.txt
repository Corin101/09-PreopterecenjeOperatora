Operatori
=========

KompleksniBroj
--------------
Primjer preopterećenja binarnog operatora + (zbrajanje) i - (oduzimanje) te unarnog operatora - (promjena predznaka)


Inkrement
---------
Preopterećenje operator inkrementiranja. Operator inkrementiranja mora promijeniti internu
vrijednost objekta i kao rezultat vratiti uvećani objekt (prefiksni inkrement). Postfiksni
operator se ne može eksplicitno definirati - njega realizira .NET Framework kroz poziv
prefiksnog inkrementa.

Uočiti kako za referentne tipove nema razlike između pre- i post-operatora jer se pridružuju
reference.


Racionalni1
-----------
Operator konstruktora pretvorbe: Racionalni se stvara iz tipa double.


Racionalni2
-----------
Definiranje operatora implicitne pretvorbe (iz long i iz double)


Racionalni3
-----------
Definiranje operatora eksplicitne pretvorbe (u long, odnosno u double) te ToXXX metoda.
